# Python To-Do List

## Project Overview

This project is a menu-driven To-Do List application developed using Python as part of the DecodeLabs Python Programming Internship.

## Features

* Add Tasks
* View Tasks
* Search Tasks
* Mark Tasks as Completed
* Delete Tasks

## Technologies Used

* Python
* Lists
* Dictionaries
* Functions
* Loops

## Learning Outcome

This project helped me understand Python fundamentals and task management logic through hands-on implementation.
